Ezra

A native app to replace PeopleSoft's Student Center from the 1800s.

Download .zip Download .tar.gz View on GitHub

Ezra is a native application for Mac and Windows that replaces Cornell's StudentCenter:

Ezra lets you do all of these things from the same dashboard:

  • Check your schedule in sweet calendar format
  • See dining hall hours and menus
  • See your ID number, ID image, and advisor
  • Review your remaining BRBs

And it's really fast!
